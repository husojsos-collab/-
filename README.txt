بوت لونا - نسخة Baileys
========================

قبل التشغيل:
1) npm install
2) انسخ مجلد images/ (اللي فيه xp.jpg، الجيفات، الفيديو) من المشروع القديم لجوه هذا المشروع
3) اعمل نسخة من .env.example باسم .env واملى المفاتيح (GROQ_API_KEY, GEMINI_API_KEY)
4) node index.js
5) افتح الرابط اللي يطلعلك بالتيرمنال (أو رابط Railway/Render) واربط برقم الهاتف أو QR

XP والترحيب/الوداع محذوفين بالكامل بطلب المستخدم.
